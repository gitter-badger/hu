# hu.js


## Usage

### Node.js module

```
$ npm install hu
```

### Browser

Via Bower package manager
```
$ bower install hu
```
Or load the script remotely (just for testing or development)
```html
<script src="//rawgithub.com/h2non/hu.js/master/hu.js"></script>
```
